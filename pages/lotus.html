Overview

I have been collaborating with Systematic to update the Lotus Entertainment website. We set them up with a beautifully designed, responsive website complete with a CMS that allowed the site owners to manage all aspects of the website.

I served as the lead back-end developer on this project. Some of my responsibilities included setting up a secure database environment, integrating with libraries such as PHPMailer, and setting up SEO-friendly URL routing.

Key Features

• Lead developer on the construction of the new Lotus website

• Setup and maintained the backend infrastructure of the website using a PHP framework

• Worked on a custom CMS that allowed the client to update the website with new films, press releases, and festivals
